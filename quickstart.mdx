# Quickstart Guide

This guide will walk you through creating your first interactive CLI application with Ink. You'll build a simple counter that displays tests passed, and then an interactive app that responds to keyboard input.

## Prerequisites

Make sure you have [Ink installed](/installation) in your project before continuing.

## Your First Ink App

Let's start with a simple example - a counter that updates every 100ms. This example is taken directly from the Ink repository.

<Steps>

<Step title="Create your application file">
  Create a new file called `counter.jsx`:

  ```jsx counter.jsx
  import React, {useState, useEffect} from 'react';
  import {render, Text} from 'ink';

  const Counter = () => {
    const [counter, setCounter] = useState(0);

    useEffect(() => {
      const timer = setInterval(() => {
        setCounter(previousCounter => previousCounter + 1);
      }, 100);

      return () => {
        clearInterval(timer);
      };
    }, []);

    return <Text color="green">{counter} tests passed</Text>;
  };

  render(<Counter />);
  ```

  <Note>
    This example uses familiar React patterns: `useState` for state management and `useEffect` for side effects.
  </Note>
</Step>

<Step title="Run your application">
  Execute your Ink app with Node.js:

  ```bash
  node counter.jsx
  ```

  You'll see a green counter that increments every 100ms, showing the number of tests passed.

  <Tip>
    Press **Ctrl+C** to exit the application at any time.
  </Tip>
</Step>

<Step title="Understand the code">
  Let's break down what's happening:

  - **`useState(0)`** - Creates state to track the counter value
  - **`useEffect(() => {...}, [])`** - Sets up a timer when the component mounts
  - **`setInterval`** - Updates the counter every 100ms
  - **`<Text color="green">`** - Renders colored text to the terminal
  - **`render(<Counter />)`** - Mounts the React component to the terminal

  The cleanup function returned from `useEffect` ensures the timer is cleared when the component unmounts.
</Step>

</Steps>

## Building an Interactive CLI

Now let's create something more interactive - a robot face you can move with arrow keys.

<Steps>

<Step title="Create an interactive app">
  Create a new file called `robot.jsx`:

  ```jsx robot.jsx
  import React, {useState} from 'react';
  import {render, useInput, useApp, Box, Text} from 'ink';

  const Robot = () => {
    const {exit} = useApp();
    const [x, setX] = useState(1);
    const [y, setY] = useState(1);

    useInput((input, key) => {
      if (input === 'q') {
        exit();
      }

      if (key.leftArrow) {
        setX(Math.max(1, x - 1));
      }

      if (key.rightArrow) {
        setX(Math.min(20, x + 1));
      }

      if (key.upArrow) {
        setY(Math.max(1, y - 1));
      }

      if (key.downArrow) {
        setY(Math.min(10, y + 1));
      }
    });

    return (
      <Box flexDirection="column">
        <Text>Use arrow keys to move the face. Press "q" to exit.</Text>
        <Box height={12} paddingLeft={x} paddingTop={y}>
          <Text>^_^</Text>
        </Box>
      </Box>
    );
  };

  render(<Robot />);
  ```

  <Note>
    This example introduces new Ink concepts: `useInput` for keyboard handling, `useApp` for app lifecycle control, and `Box` for layout.
  </Note>
</Step>

<Step title="Run the interactive app">
  ```bash
  node robot.jsx
  ```

  Try it out:
  - Use **arrow keys** to move the robot face around
  - Press **q** to quit the application

  The robot face (`^_^`) will move within a bounded area as you press the arrow keys.
</Step>

<Step title="Learn the new concepts">
  This example introduces several important Ink concepts:

  **`useInput()`** - A hook that handles keyboard input:
  ```jsx
  useInput((input, key) => {
    // input: the character typed
    // key: object with boolean flags for special keys
    if (input === 'q') {
      // Handle regular key press
    }
    if (key.leftArrow) {
      // Handle arrow keys
    }
  });
  ```

  **`useApp()`** - Provides app lifecycle methods:
  ```jsx
  const {exit} = useApp();
  exit(); // Cleanly exit the application
  ```

  **`<Box>`** - Layout component with Flexbox properties:
  ```jsx
  <Box flexDirection="column" paddingLeft={2} paddingTop={1}>
    {/* Content */}
  </Box>
  ```
</Step>

</Steps>

## Using Layouts with Flexbox

Ink uses Yoga, a Flexbox layout engine, so you can use CSS-like properties for layouts.

<Steps>

<Step title="Create a layout example">
  ```jsx layout.jsx
  import React from 'react';
  import {render, Box, Text} from 'ink';

  const Layout = () => (
    <Box flexDirection="column" padding={1}>
      <Box borderStyle="round" borderColor="cyan" padding={1}>
        <Text bold>Header Section</Text>
      </Box>
      
      <Box marginTop={1} gap={2}>
        <Box borderStyle="single" padding={1} flexGrow={1}>
          <Text>Left Panel</Text>
        </Box>
        <Box borderStyle="single" padding={1} flexGrow={2}>
          <Text>Right Panel (2x wider)</Text>
        </Box>
      </Box>
    </Box>
  );

  render(<Layout />);
  ```

  <Tip>
    Think of every `<Box>` as having `display: flex` in CSS. Use Flexbox properties like `flexDirection`, `justifyContent`, `alignItems`, and `flexGrow`.
  </Tip>
</Step>

<Step title="Run the layout example">
  ```bash
  node layout.jsx
  ```

  You'll see a structured layout with borders, demonstrating how Flexbox works in the terminal.
</Step>

</Steps>

## Common Patterns

### Displaying Static Lists

Use the `<Static>` component for content that doesn't change after rendering:

```jsx
import React, {useState, useEffect} from 'react';
import {Box, Text, render, Static} from 'ink';

const TestRunner = () => {
  const [tests, setTests] = useState([]);

  useEffect(() => {
    let completedTests = 0;
    let timer;

    const run = () => {
      if (completedTests++ < 10) {
        setTests(previousTests => [
          ...previousTests,
          {
            id: previousTests.length,
            title: `Test #${previousTests.length + 1}`
          }
        ]);
        timer = setTimeout(run, 100);
      }
    };

    run();
    return () => clearTimeout(timer);
  }, []);

  return (
    <>
      <Static items={tests}>
        {test => (
          <Box key={test.id}>
            <Text color="green">✔ {test.title}</Text>
          </Box>
        )}
      </Static>

      <Box marginTop={1}>
        <Text dimColor>Completed tests: {tests.length}</Text>
      </Box>
    </>
  );
};

render(<TestRunner />);
```

<Note>
  `<Static>` is perfect for logs, completed tasks, or any output that accumulates over time. New items appear above the live-updating content.
</Note>

### Handling User Input

Build interactive menus and forms with `useInput`:

```jsx
const Menu = () => {
  const [selected, setSelected] = useState(0);
  const items = ['Option 1', 'Option 2', 'Option 3'];

  useInput((input, key) => {
    if (key.upArrow) {
      setSelected(Math.max(0, selected - 1));
    }
    if (key.downArrow) {
      setSelected(Math.min(items.length - 1, selected + 1));
    }
  });

  return (
    <Box flexDirection="column">
      {items.map((item, index) => (
        <Text key={item} color={index === selected ? 'blue' : 'white'}>
          {index === selected ? '> ' : '  '}{item}
        </Text>
      ))}
    </Box>
  );
};
```
