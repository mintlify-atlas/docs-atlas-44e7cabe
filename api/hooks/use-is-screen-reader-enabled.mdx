---
title: useIsScreenReaderEnabled
description: Hook for detecting screen reader availability
---

A React hook that returns whether a screen reader is enabled. This is useful when you want to render different output for screen readers.

## Signature

```typescript
const useIsScreenReaderEnabled = () => boolean
```

## Parameters

This hook takes no parameters.

## Return Value

Returns a `boolean` indicating whether a screen reader is currently enabled.

## Example

```tsx
import {useIsScreenReaderEnabled, Text} from 'ink';

const Example = () => {
  const isScreenReaderEnabled = useIsScreenReaderEnabled();

  return (
    <Text>
      {isScreenReaderEnabled
        ? 'Screen reader detected: Providing enhanced accessibility'
        : 'Standard output mode'}
    </Text>
  );
};
```
