---
title: Box
description: Essential layout component for building CLI interfaces
---

`<Box>` is an essential Ink component to build your layout. It's like `<div style="display: flex">` in the browser.

## Component Signature

```tsx
import {Box} from 'ink';

function Component() {
  return (
    <Box flexDirection="column" padding={1}>
      {/* Your content */}
    </Box>
  );
}
```

## Props

### Layout & Flexbox

<ParamField path="flexDirection" type="'row' | 'column' | 'row-reverse' | 'column-reverse'">
  It establishes the main-axis, thus defining the direction flex items are placed in the flex container.
  See [flex-direction](https://css-tricks.com/almanac/properties/f/flex-direction/).
</ParamField>

<ParamField path="flexGrow" type="number">
  This property defines the ability for a flex item to grow if necessary.
  See [flex-grow](https://css-tricks.com/almanac/properties/f/flex-grow/).
</ParamField>

<ParamField path="flexShrink" type="number">
  It specifies the "flex shrink factor", which determines how much the flex item will shrink relative to the rest of the flex items in the flex container when there isn't enough space on the row.
  See [flex-shrink](https://css-tricks.com/almanac/properties/f/flex-shrink/).
</ParamField>

<ParamField path="flexBasis" type="number | string">
  It specifies the initial size of the flex item, before any available space is distributed according to the flex factors.
  See [flex-basis](https://css-tricks.com/almanac/properties/f/flex-basis/).
</ParamField>

<ParamField path="flexWrap" type="'nowrap' | 'wrap' | 'wrap-reverse'">
  It defines whether the flex items are forced in a single line or can be flowed into multiple lines. If set to multiple lines, it also defines the cross-axis which determines the direction new lines are stacked in.
  See [flex-wrap](https://css-tricks.com/almanac/properties/f/flex-wrap/).
</ParamField>

<ParamField path="alignItems" type="'flex-start' | 'center' | 'flex-end' | 'stretch' | 'baseline'">
  The align-items property defines the default behavior for how items are laid out along the cross axis (perpendicular to the main axis).
  See [align-items](https://css-tricks.com/almanac/properties/a/align-items/).
</ParamField>

<ParamField path="alignSelf" type="'flex-start' | 'center' | 'flex-end' | 'auto' | 'stretch' | 'baseline'">
  It makes possible to override the align-items value for specific flex items.
  See [align-self](https://css-tricks.com/almanac/properties/a/align-self/).
</ParamField>

<ParamField path="alignContent" type="'flex-start' | 'flex-end' | 'center' | 'stretch' | 'space-between' | 'space-around' | 'space-evenly'">
  It defines the alignment along the cross axis when there are multiple lines of flex items (when using flex-wrap).
  See [align-content](https://css-tricks.com/almanac/properties/a/align-content/).
</ParamField>

<ParamField path="justifyContent" type="'flex-start' | 'flex-end' | 'space-between' | 'space-around' | 'space-evenly' | 'center'">
  It defines the alignment along the main axis.
  See [justify-content](https://css-tricks.com/almanac/properties/j/justify-content/).
</ParamField>

### Dimensions

<ParamField path="width" type="number | string">
  Width of the element in spaces. You can also set it as a percentage, which will calculate the width based on the width of the parent element.
</ParamField>

<ParamField path="height" type="number | string">
  Height of the element in lines (rows). You can also set it as a percentage, which will calculate the height based on the height of the parent element.
</ParamField>

<ParamField path="minWidth" type="number | string">
  Sets a minimum width of the element.
  Percentages aren't supported yet; see https://github.com/facebook/yoga/issues/872.
</ParamField>

<ParamField path="minHeight" type="number | string">
  Sets a minimum height of the element in lines (rows). You can also set it as a percentage, which will calculate the minimum height based on the height of the parent element.
</ParamField>

<ParamField path="maxWidth" type="number | string">
  Sets a maximum width of the element.
  Percentages aren't supported yet; see https://github.com/facebook/yoga/issues/872.
</ParamField>

<ParamField path="maxHeight" type="number | string">
  Sets a maximum height of the element in lines (rows). You can also set it as a percentage, which will calculate the maximum height based on the height of the parent element.
</ParamField>

<ParamField path="aspectRatio" type="number">
  Defines the aspect ratio (width/height) for the element.
  
  Use it with at least one size constraint (`width`, `height`, `minHeight`, or `maxHeight`) so Ink can derive the missing dimension.
</ParamField>

## Example

```tsx
import {Box, Text} from 'ink';

function Example() {
  return (
    <Box
      flexDirection="column"
      padding={2}
      borderStyle="round"
      borderColor="green"
    >
      <Text>Hello from Box!</Text>
    </Box>
  );
}
```
