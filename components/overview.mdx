Ink provides a set of built-in components that serve as the foundation for creating beautiful command-line interfaces. These components are inspired by HTML/CSS and work together using Flexbox layouts.

## Core Components

Ink's component library is intentionally minimal, providing just the essential building blocks you need:

## Design Philosophy

Ink's components follow these principles:

### React-First

All components are React components that work just like their browser counterparts. If you know React, you already know Ink.

```jsx
import React from 'react';
import {render, Text, Box} from 'ink';

const App = () => (
  <Box flexDirection="column">
    <Text color="green">Hello World</Text>
  </Box>
);

render(<App />);
```

### Flexbox Layouts

Every element uses Flexbox for layout, powered by [Yoga](https://github.com/facebook/yoga). Think of every component as having `display: flex` by default.

```jsx
<Box justifyContent="space-between">
  <Text>Left</Text>
  <Text>Right</Text>
</Box>
```

### Text Wrapping

All text must be wrapped in a `<Text>` component. This ensures proper styling and layout calculations.

```jsx
{/* ✅ Correct */}
<Box>
  <Text>Hello</Text>
</Box>

{/* ❌ Wrong */}
<Box>
  Hello
</Box>
```

## Component Composition

Components are designed to be composed together:

```jsx
import {Box, Text, Spacer} from 'ink';

const Header = () => (
  <Box borderStyle="round" padding={1}>
    <Text bold>My App</Text>
    <Spacer />
    <Text dimColor>v1.0.0</Text>
  </Box>
);
```
