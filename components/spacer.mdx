The `<Spacer>` component is a flexible space that expands along the major axis of its containing layout. It's a convenient shortcut for creating space between elements without manually setting flex properties.

<Note>
  `<Spacer>` is essentially a `<Box>` with `flexGrow={1}`, but more semantic and easier to read.
</Note>

## Basic Usage

```jsx
import {render, Box, Text, Spacer} from 'ink';

const App = () => (
  <Box>
    <Text>Left</Text>
    <Spacer />
    <Text>Right</Text>
  </Box>
);

render(<App />);
// Output: Left                    Right
```

## How It Works

`<Spacer>` expands to fill all available space in the direction of the flex layout:

- In `flexDirection="row"` (default): Pushes elements horizontally
- In `flexDirection="column"`: Pushes elements vertically

```jsx
// Horizontal spacing (default)
<Box>
  <Text>A</Text>
  <Spacer />
  <Text>B</Text>
</Box>
// A                              B

// Vertical spacing
<Box flexDirection="column" height={10}>
  <Text>Top</Text>
  <Spacer />
  <Text>Bottom</Text>
</Box>
// Top
//
//
//
//
// Bottom
```

## Real-World Examples

### Header with Version

```jsx
import {Box, Text, Spacer} from 'ink';

const Header = ({title, version}) => (
  <Box>
    <Text bold color="blue">{title}</Text>
    <Spacer />
    <Text dimColor>v{version}</Text>
  </Box>
);

const App = () => (
  <Header title="My CLI App" version="1.0.0" />
);
// Output: My CLI App                v1.0.0
```

### Status Line

```jsx
const StatusLine = ({status, time}) => (
  <Box>
    <Text color={status === 'success' ? 'green' : 'red'}>
      {status === 'success' ? '✓' : '✗'}
    </Text>
    <Text> {status}</Text>
    <Spacer />
    <Text dimColor>{time}</Text>
  </Box>
);

const App = () => (
  <StatusLine status="success" time="2.3s" />
);
// Output: ✓ success                 2.3s
```

### Navigation Bar

```jsx
const NavBar = ({left, center, right}) => (
  <Box>
    <Text>{left}</Text>
    <Spacer />
    <Text bold>{center}</Text>
    <Spacer />
    <Text>{right}</Text>
  </Box>
);

const App = () => (
  <NavBar
    left="← Back"
    center="Settings"
    right="Next →"
  />
);
// Output: ← Back        Settings        Next →
```

### Vertical Centering

```jsx
const CenteredDialog = ({message}) => (
  <Box flexDirection="column" height={20}>
    <Spacer />
    <Box justifyContent="center">
      <Text>{message}</Text>
    </Box>
    <Spacer />
  </Box>
);
// Vertically centers the message
```

### Footer

```jsx
const Footer = () => (
  <Box borderStyle="round" padding={1}>
    <Text dimColor>Press Ctrl+C to exit</Text>
    <Spacer />
    <Text dimColor>Help: ?</Text>
  </Box>
);
```

## Common Patterns

### Multiple Spacers

You can use multiple spacers to distribute space evenly:

```jsx
<Box>
  <Text>A</Text>
  <Spacer />
  <Text>B</Text>
  <Spacer />
  <Text>C</Text>
</Box>
// A          B          C
```

### With Fixed-Width Elements

```jsx
<Box>
  <Box width={10}>
    <Text>Fixed</Text>
  </Box>
  <Spacer />
  <Text>Flexible</Text>
</Box>
// Fixed                    Flexible
```

### Pushing to Edges

```jsx
// Push to right
<Box>
  <Spacer />
  <Text>Right-aligned</Text>
</Box>

// Push to left (default, but explicit)
<Box>
  <Text>Left-aligned</Text>
  <Spacer />
</Box>

// Push to bottom
<Box flexDirection="column" height={10}>
  <Spacer />
  <Text>Bottom-aligned</Text>
</Box>
```

### Toolbar Layout

```jsx
const Toolbar = ({tools, info}) => (
  <Box>
    {tools.map((tool, i) => (
      <Box key={i} marginRight={1}>
        <Text>{tool}</Text>
      </Box>
    ))}
    <Spacer />
    <Text dimColor>{info}</Text>
  </Box>
);

const App = () => (
  <Toolbar
    tools={['New', 'Open', 'Save']}
    info="Modified"
  />
);
// Output: New Open Save           Modified
```

## Spacer vs Alternatives

### Spacer vs flexGrow

```jsx
// Using Spacer (recommended)
<Box>
  <Text>Left</Text>
  <Spacer />
  <Text>Right</Text>
</Box>

// Using flexGrow (equivalent)
<Box>
  <Text>Left</Text>
  <Box flexGrow={1} />
  <Text>Right</Text>
</Box>
```

### Spacer vs justifyContent

```jsx
// Using Spacer
<Box>
  <Text>Left</Text>
  <Spacer />
  <Text>Right</Text>
</Box>

// Using justifyContent="space-between"
<Box justifyContent="space-between">
  <Text>Left</Text>
  <Text>Right</Text>
</Box>
```

<Info>
  Use `<Spacer>` when you want explicit control over which elements get space between them. Use `justifyContent` when all direct children should be spaced evenly.
</Info>

### Spacer vs Margin

```jsx
// Using Spacer (flexible)
<Box>
  <Text>A</Text>
  <Spacer />
  <Text>B</Text>
</Box>
// Space adjusts based on available width

// Using margin (fixed)
<Box>
  <Text>A</Text>
  <Box marginLeft={10}>
    <Text>B</Text>
  </Box>
</Box>
// Always 10 spaces, may overflow
```

## Layout Examples

### Three-Column Layout

```jsx
const ThreeColumn = ({left, center, right}) => (
  <Box>
    <Box width="33%"}>
      <Text>{left}</Text>
    </Box>
    <Box width="34%" justifyContent="center">

    <Spacer />
    <Text>{right}</Text>
```

### Card with Actions

```jsx
const Card = ({title, content, actions}) => (
  <Box
    flexDirection="column"
    borderStyle="round"
    padding={1}
  >
    <Text bold>{title}</Text>
    <Box marginY={1}>
      <Text>{content}</Text>
    </Box>
    <Spacer />
    <Box>

        </Box>
      ))}

  </Box>
);
```

## Vertical Layouts

### Modal Dialog

```jsx
const Modal = ({title, message, buttons}) => (
  <Box
    flexDirection="column"
    borderStyle="round"
    padding={1}
    height={15}
  >
    <Text bold>{title}</Text>
    
    <Box marginY={1}>
      <Text>{message}</Text>
    </Box>
    
    <Spacer />
    
    <Box justifyContent="flex-end" columnGap={1}>
      {buttons.map((btn, i) => (
        <Text key={i} backgroundColor="blue" bold>
          {' '}{btn}{' '}
        </Text>
      ))}
    </Box>
  </Box>
);
```

```jsx
    <Box height="50%" borderStyle="single" borderBottom={false}>
      {top}
    </Box>
    <Box height="50%" borderStyle="single">
      {bottom}
    </Box>
  </Box>
);
```

## Tips and Best Practices

<Info>
  **When to use Spacer:**
  - Need flexible space that adapts to container size
  - Want semantic, readable code
  - Building responsive layouts
  - Creating navigation bars, headers, footers
</Info>

<Warning>
  **When NOT to use Spacer:**
  - Need fixed spacing (use `margin` instead)
  - All children should be evenly spaced (use `justifyContent` instead)
  - Need precise control over spacing (use `gap` or `margin` instead)
</Warning>

### Performance

`<Spacer>` is just a `<Box>` component, so there's no performance difference:

```jsx
// These are identical in performance
<Spacer />
<Box flexGrow={1} />
```
