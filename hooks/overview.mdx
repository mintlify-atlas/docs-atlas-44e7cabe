## Hook Categories

### Input Handling

- **[useInput](/hooks/use-input)** - Capture keyboard input with support for special keys, modifiers, and key combinations
- **[usePaste](/hooks/use-paste)** - Handle pasted text separately from typed input using bracketed paste mode

### Focus Management

- **[useFocus](/hooks/use-focus)** - Make individual components focusable and respond to Tab navigation
- **[useFocusManager](/hooks/use-focus-manager)** - Programmatically control focus across your entire application

### App Lifecycle

- **[useApp](/hooks/use-app)** - Exit the application gracefully and wait for render flushes

### Terminal I/O

- **[useStdin](/hooks/use-stdin)** - Access stdin stream and control raw mode for custom input handling
- **[useStdout](/hooks/use-stdout)** - Write to stdout outside of React's rendering pipeline
- **[useStderr](/hooks/use-stderr)** - Write error messages and logs to stderr

### Layout & Measurement

- **[useBoxMetrics](/hooks/use-box-metrics)** - Track the computed dimensions and position of box elements
- **[useWindowSize](/hooks/use-window-size)** - Respond to terminal window resizing
- **[useCursor](/hooks/use-cursor)** - Position the cursor for IME (Input Method Editor) support

### Accessibility

- **[useIsScreenReaderEnabled](/hooks/use-is-screen-reader-enabled)** - Render different output for screen reader users

## Common Patterns

### Building an Interactive Input

Combine `useInput`, `usePaste`, and `useFocus` to create rich input components:

```jsx
import {useState} from 'react';
import {Box, Text, useInput, usePaste, useFocus} from 'ink';

const TextInput = () => {
  const [value, setValue] = useState('');
  const {isFocused} = useFocus();

  useInput((input, key) => {
    if (key.backspace) {
      setValue(v => v.slice(0, -1));
    } else if (!key.return && input) {
      setValue(v => v + input);
    }
  }, {isActive: isFocused});

  usePaste((text) => {
    setValue(v => v + text);
  }, {isActive: isFocused});

  return (
    <Box>
      <Text>{isFocused ? '> ' : '  '}</Text>
      <Text>{value}</Text>
    </Box>
  );
};
```

### Responsive Layout

Use `useWindowSize` to adapt your layout to terminal dimensions:

```jsx
import {Box, Text, useWindowSize} from 'ink';

const ResponsiveLayout = () => {
  const {columns, rows} = useWindowSize();
  const isNarrow = columns < 80;

  return (
    <Box flexDirection={isNarrow ? 'column' : 'row'}>
      <Text>Adapts to terminal size: {columns}x{rows}</Text>
    </Box>
  );
};
```

### Graceful Exit

Use `useApp` to handle cleanup before exiting:

```jsx
import {useEffect} from 'react';
import {useApp, useInput, Text} from 'ink';

const MyApp = () => {
  const {exit} = useApp();

  useInput((input) => {
    if (input === 'q') {
      // Perform cleanup, then exit
      exit();
    }
  });

  return <Text>Press 'q' to quit</Text>;
};
```
