---
title: 'useInput'
description: 'Handle keyboard input from users in your CLI application'
---

`useInput` is a React hook that handles user keyboard input. It's a convenient alternative to manually listening for data events on stdin. The hook calls your handler for each input event, providing both the raw input and parsed key information.

## Import

```jsx
import {useInput} from 'ink';
```

## Signature

```typescript
useInput(
  inputHandler: (input: string, key: Key) => void,
  options?: {isActive?: boolean}
): void
```

## Parameters

## Return Value

`void` - This hook doesn't return anything.

## Examples

### Basic Input Handling

Handle simple keyboard shortcuts:

```jsx
import {useInput, useApp, Text} from 'ink';

const App = () => {
  const {exit} = useApp();

  useInput((input, key) => {
    if (input === 'q') {
      exit();
    }
  });

  return <Text>Press 'q' to quit</Text>;
};
```

### Arrow Key Navigation

Move an element around the screen using arrow keys:

```jsx
import {useState} from 'react';
import {Box, Text, useInput, useApp} from 'ink';

const Robot = () => {
  const {exit} = useApp();
  const [x, setX] = useState(1);
  const [y, setY] = useState(1);

  useInput((input, key) => {
    if (input === 'q') {
      exit();
    }

    if (key.leftArrow) {
      setX(Math.max(1, x - 1));
    }

    if (key.rightArrow) {
      setX(Math.min(20, x + 1));
    }

    if (key.upArrow) {
      setY(Math.max(1, y - 1));
    }

    if (key.downArrow) {
      setY(Math.min(10, y + 1));
    }
  });

  return (
    <Box flexDirection="column">
      <Text>Use arrow keys to move the face. Press "q" to exit.</Text>
      <Box height={12} paddingLeft={x} paddingTop={y}>
        <Text>^_^</Text>
      </Box>
    </Box>
  );
};
```

### Text Input Field

Build a simple text input component:

```jsx
import {useState} from 'react';
import {Box, Text, useInput} from 'ink';

const TextInput = ({onSubmit}) => {
  const [value, setValue] = useState('');

  useInput((input, key) => {
    if (key.return) {
      onSubmit(value);
      setValue('');
    } else if (key.backspace) {
      setValue(value.slice(0, -1));
    } else if (!key.ctrl && !key.meta && input) {
      setValue(value + input);
    }
  });

  return (
    <Box>
      <Text>Input: {value}</Text>
    </Box>
  );
};
```

### Conditional Input Handling

Use the `isActive` option to control when input is captured:

```jsx
import {useState} from 'react';
import {Box, Text, useInput} from 'ink';

const Modal = ({isOpen, onClose}) => {
  useInput((input) => {
    if (input === 'q') {
      onClose();
    }
  }, {isActive: isOpen});

  if (!isOpen) return null;

  return (
    <Box borderStyle="round" padding={1}>
      <Text>Modal is open. Press 'q' to close.</Text>
    </Box>
  );
};
```

### Ctrl Key Combinations

Detect Ctrl+key combinations:

```jsx
import {useInput, Text} from 'ink';

const App = () => {
  useInput((input, key) => {
    if (key.ctrl) {
      switch (input) {
        case 's':
          console.log('Save');
          break;
        case 'o':
          console.log('Open');
          break;
        case 'n':
          console.log('New');
          break;
      }
    }
  });

  return <Text>Try Ctrl+S, Ctrl+O, or Ctrl+N</Text>;
};
```

## Notes

<Note>
  If the user pastes text and it's more than one character, the callback will be called only once with the entire pasted string as `input`. For better paste handling, consider using [`usePaste`](/hooks/use-paste) alongside `useInput`.
</Note>

<Note>
  When `useInput` is active, the terminal is automatically set to raw mode. This is restored when the component unmounts.
</Note>

<Warning>
  If your app has `exitOnCtrlC` enabled (the default), pressing Ctrl+C will exit the app before your `useInput` handler is called.
</Warning>
