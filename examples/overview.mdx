## Running Examples Locally

All examples are available in the Ink repository:

```bash
git clone https://github.com/vadimdemedes/ink.git
cd ink/examples
```

Each example can be run directly:

```bash
# Using npm
npm run example examples/counter

# Or run directly with tsx
npx tsx examples/counter/counter.tsx
```
